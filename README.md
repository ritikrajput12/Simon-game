# Simon Game

A simple Simon Says memory game built using HTML, CSS, and JavaScript.
The game tests how well you can remember and repeat a growing sequence of colors.

Live Demo: https://simon-game-neon-psi.vercel.app

---

## Overview

This project is a basic implementation of the classic Simon game.
The goal is to follow the sequence shown on the screen and repeat it correctly.
With each level, the sequence gets longer and more challenging.

---

## How to play

* Press any key to start the game
* Watch the sequence of colors carefully
* Click the buttons in the same order
* Each level adds one new step
* If you make a mistake, the game ends

---

## Features

* Increasing difficulty with each level
* Visual feedback using button animations
* Game over screen with score
* Works on desktop and mobile

---

## Tech Stack

* HTML
* CSS
* JavaScript
